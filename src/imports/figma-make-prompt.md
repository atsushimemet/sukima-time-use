# Figma Make Prompt

Design a **modern mobile productivity app UI** for iOS called **Time Slotter**.

The concept is **planning how to use small moments of time in daily life**.

Users predefine actions for specific situations.

Examples:

* Green train seat → Read a physical book
* Crowded train → Read Kindle
* Walking → Listen to podcast
* Café → Write notes

The app helps users **decide in advance what to do in each situation**, reducing decision fatigue and turning small moments into productive time.

---

## Design Style

Create a **modern minimal productivity app UI**.

Style inspiration:

* Apple Human Interface Guidelines
* Notion soft cards
* Calm productivity apps

Visual characteristics:

* soft rounded cards
* generous white space
* calm neutral background
* subtle shadows
* minimal text
* focus on clarity and calmness

Color palette:

* off-white background
* soft green primary accent
* pale blue secondary accent
* light gray UI elements

No bright or aggressive colors.

Tone: **calm, thoughtful, intelligent**

---

## Screens to design

### 1. Home Screen

Show the **current situation and recommended action**.

Example layout:

Top section:

Current Situation Card

Icon + label

"Crowded Train"

Main action card below:

→ Read Kindle

Large typography

Below:

Upcoming situations

Small cards:

* Green Seat → Read Book
* Walking → Podcast
* Café → Write

---

### 2. Situation Setup Screen

Users can define situations.

Example list:

* Crowded Train
* Green Car
* Walking
* Café
* Waiting Time

Each situation displayed as **icon cards**.

Tap to assign an action.

---

### 3. Action Selection Screen

User selects action for a situation.

Examples:

* Read Physical Book
* Read Kindle
* Write Notes
* Listen to Podcast
* Review Notes

Design with **clean selectable cards**.

---

### 4. Weekly Review Screen

Encourage reflection.

Simple dashboard:

This week:

Reading: 2h
Writing: 45m
Podcast: 1h

Minimal charts.

---

## UX Principles

Design for:

* extremely low cognitive load
* one clear action per screen
* calm productivity
* minimal distractions

The interface should feel like a **quiet thinking tool**, not a busy task manager.

---

## Components

Include:

* icon cards
* soft buttons
* bottom tab navigation
* subtle micro-interactions
* rounded elements
